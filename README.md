##  Что за проект
Это верстка адаптивного лендинга для дизайнера **Greg Klassen** по макету из Figma из открытых источников.  
Сайт представляет портфолио мастера мебели ручной работы, его работы и контактную информацию.

##  Какие технологии использовал
- HTML5
- CSS3 (Flexbox, Grid)
- Figma

##  Что получилось
- Полная адаптация под мобильные устройства
- Семантическая разметка
- Соответствие BEM (стандарт названий тегов, классов и тд)
- Кастомные анимации и hover-эффекты
- Форма обратной связи с валидацией полей
